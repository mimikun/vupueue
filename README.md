# vupueue

Tools to update various packages and commands using pueue. (for mimikun)

## Requirements

- [os_info](https://github.com/stanislav-tkach/os_info)
- [pueue](https://github.com/Nukesor/pueue)

## Install

```shell
git clone https://github.com/mimikun/vupueue.git
cd vupueue
make install
```
